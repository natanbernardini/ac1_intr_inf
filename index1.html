primeiro passo: placa mãe

ao apertar o botão de energia
do computador, é na qual liga
a fonte e controla a energia
para a placa mãe

segundo passo: boot

o compuador entra no
processo de boot, que
ocorre entre o momento em
que um computador é
ligado pelo usuário até a
inicialização completa do
sistema operacional

terceiro passo: input

é nessa etapa que ocorre
o processo de basic input
output system 1° auto-teste,
mais conhecido como BIOS.

quarto passo: processador

nessa etapa, é acionado
o processador da máquina
e sua função é acelerar,endereçar,
resolver ou preparar dados,
dependendo da aplicação

quinto passo: rodar um programa

quando você dá um comando
ao clicar em um elemento
na interface de um programa
ou sistema operacional,
você está basicamente iniciando
uma ação ou instruindo o
software para realizar uma
tarefa específica
